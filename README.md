# Ekranchik Modern

**Ekranchik Modern** — клиент-серверный комплекс для круглосуточного и отказоустойчивого отображения промышленного дашборда гальванической линии на выделенных мониторах.

---

## Архитектура системы

Комплекс состоит из трех основных частей:

1. **Backend**:
   * Написан на **FastAPI** (Python).
   * Собирает технологические данные по протоколу **OPC UA** и считывает отчетные файлы **Excel**.
   * Предоставляет REST API и WebSocket для клиентской части.

2. **Frontend**:
   * Современный веб-интерфейс на **React + Vite**.
   * Отображает интерактивные графики, таблицы, статусы ванн и предупреждения для операторов линии.

3. **Ekranchik Settings / Launcher**:
   * Десктопное приложение для Windows на **.NET 9 (WinForms)**.
   * Позволяет выбрать монитор (в виде больших кнопок-переключателей), указать URL бэкенда, настроить автозапуск и отслеживать пинг/доступность ссылок в реальном времени.

## Быстрый запуск (Разработка)

### Бэкенд
1. Установите Python.
2. Установите зависимости и запустите:
   ```bash
   pip install -r requirements.txt
   uvicorn app.main:app --reload --port 8000
   ```

### Фронтенд
1. Установите Node.js.
2. Установите зависимости и запустите:
   ```bash
   npm install
   npm run dev -- --port 5173
   ```

---

## Деплой в Production (Docker)

Запуск продакшен-версии бэкенда, фронтенда и проксирующего Nginx выполняется через Docker Compose:

```bash
docker compose -f docker-compose.prod.yml up -d --build
```

## Подключение сетевой папки (Mount) на Виртуальной Машине

Чтобы бэкенд в Docker-контейнере мог читать файлы отчетов Excel с общего сетевого диска Windows, этот диск нужно примонтировать к виртуальной машине.

Для этого подготовлен готовый скрипт **[mount_and_start.sh](file:///c:/Users/user/VibeCoding/ekranchik-modern/infra/mount_and_start.sh)**.

### Как запустить монтирование:
1. Запустите скрипт на вашей виртуальной машине:
   ```bash
   sudo ./mount_and_start.sh
   ```
2. Скрипт запросит учетные данные к сетевой папке, настроит автоматическое подключение диска при перезагрузке системы и запустит Docker-контейнеры.
3. Бэкенд автоматически начнет считывать отчетные таблицы из примонтированной сетевой папки.
