# Руководство по быстрому старту (Getting Started)

В этом документе описано, как подготовить окружение, установить зависимости и запустить систему **Ekranchik-Modern** локально для разработки.

## Требования к системе

1. **Python 3.10+** (необходим для бэкенда и симулятора).
2. **Node.js 18+** и **npm** (необходимы для фронтенда и запуска оркестрации).
3. **Git** (для работы с репозиторием).
4. На Linux-серверах для работы симулятора с GUI Tkinter может потребоваться пакет `xvfb`.

## Шаги установки

### Шаг 1. Клонирование и переход в проект
Перейдите в рабочую директорию проекта:
```bash
cd c:\Users\user\VibeCoding\ekranchik-modern
```

### Шаг 2. Установка зависимостей Node.js
В корне проекта выполните специальный скрипт, который установит зависимости для корня и для фронтенда:
```bash
npm run install-all
```
Этот скрипт выполнит:
1. `npm install` в корне (установит `concurrently` и `shell-quote` для запуска сервисов).
2. `npm install --prefix frontend` (установит все React-зависимости, включая Tailwind CSS, Radix UI и Vitest).

### Шаг 3. Настройка виртуального окружения Python
Для бэкенда рекомендуется использовать виртуальное окружение:
```powershell
# В Windows (PowerShell)
cd backend
python -m venv venv
.\venv\Scripts\Activate.ps1

# Установка зависимостей бэкенда
pip install -r requirements.txt
```

### Шаг 4. Настройка переменных окружения
Создайте файл `.env` в директории `backend/` на основе примера:
```powershell
Copy-Item .env.example .env
```
Откройте файл `backend/.env` и при необходимости скорректируйте параметры (например, включите или выключите симуляцию через `SIMULATION_ENABLED`).

### Шаг 5. Применение миграций базы данных
Внутри виртуального окружения бэкенда примените миграции Alembic для инициализации базы данных SQLite:
```bash
alembic upgrade head
```

## Запуск проекта

### Быстрый запуск всех компонентов (Бэкенд + Фронтенд + Симулятор)
Из корня проекта выполните команду:
```bash
npm run dev
```
Команда использует утилиту `concurrently` для параллельного запуска:
1. **Бэкенда** (FastAPI) на порту `8000`.
2. **Фронтенда** (Vite + React) на порту `5173`.
3. **Симулятора OPC UA** (Tkinter GUI).

### Раздельный запуск компонентов
Если вам нужно запустить только определенные сервисы:

* **Запуск только Фронтенда**:
  ```bash
  npm run dev:frontend
  ```
  *(или `npm run dev --prefix frontend`)*

* **Запуск только Бэкенда**:
  ```bash
  npm run dev:backend
  ```
  *(выполняет `cd backend && python -m uvicorn app.main:app --host 0.0.0.0 --port 8000`)*

* **Запуск только Симулятора OPC UA**:
  ```bash
  npm run dev:simulator
  ```
  *(выполняет `cd backend && python opcua_server_simulator.py`)*

## Список скриптов в `package.json`

Корневой файл [package.json](file:///c:/Users/user/VibeCoding/ekranchik-modern/package.json) предоставляет следующие команды:
* `install-all`: Установка Node.js зависимостей для всего проекта.
* `dev`: Запуск бэкенда, фронтенда и симулятора параллельно.
* `dev:frontend`: Запуск локального сервера разработки React.
* `dev:backend`: Запуск FastAPI сервера через Uvicorn.
* `dev:simulator`: Запуск имитатора OPC UA.
